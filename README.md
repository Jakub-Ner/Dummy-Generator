## Generator wartości na laboratoria z Baz Danych:

- Program generuje tabele w formacie .csv.  
- Wartości są:
  - zgodne z wytycznymi
  - zawierają polskie znaki 
  - zgodne z rzeczywistością
  - pozbawione błędów ortograficznych
  
## Instalacja

```bash
pip install factory-boy==3.2.1  
```

### Uruchomienie

```bash
python3 main.py
```
